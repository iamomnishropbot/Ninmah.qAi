### Advanced Setup (All Platforms)

If you prefer manual setup or are on macOS/Linux, follow these steps:

### Prerequisites

- **Node.js** (v18+) and npm
- **Python** (v3.9+) and pip
- **Common Lisp** implementation (SBCL recommended)
- **OpenRouter API key** ([Get one here](https://openrouter.ai/keys))

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/iamomnishropbot/Ninmah.qAi.git
   cd Ninmah.qAi
   ```

2. **Set up environment variables:**
   ```bash
   cp .env.example .env
   # Edit .env and add your API keys and tokens
   ```

3. **Install backend dependencies:**
   ```bash
   cd backend/python
   pip install -r requirements.txt
   ```

4. **Install frontend dependencies:**
   ```bash
   cd frontend
   npm install
   ```

5. **Start the LISP consciousness engine:**
   ```bash
   cd backend/lisp
   sbcl --load consciousness.lisp
   ```

6. **Start the Python backend:**
   ```bash
   cd backend/python
   uvicorn main:app --reload --port 8000
   ```

7. **Start the frontend:**
   ```bash
   cd frontend
   npm run dev
   ```

8. **Visit http://localhost:5173** and authenticate.

For detailed setup instructions, see [SETUP.md](docs/SETUP.md).
